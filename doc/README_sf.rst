Obtaining the source code
=========================

The sources are hosted on
`github.com
<https://github.com/dkriegner/xrayutilities>`_ in are made available
as tarballs from time to time on
`github
<https://github.com/dkriegner/xrayutilities/releases>`_ and
`sourceforge.net
<https://sourceforge.net/projects/xrayutilities/files/>`_

Download the latest tarball or use:

    git clone https://github.com/dkriegner/xrayutilities.git


to clone the git repository. If you would like to have commit rights
contact one of the administrators.


Installation
============

xrayutilities can be installed using several methods:

1. Using pip:

    pip install xrayutilities

2. Using conda:

    conda install -c conda-forge xrayutilities

3. On Debian-based Linux (Debian, Ubuntu, ...):

    sudo apt install python3-xrayutilities

4. From source:

    pip install .

A few notes on how to install on various systems can be found in the README
shipped with the package's sources or on the `webpage <https://xrayutilities.sourceforge.io>`_.
